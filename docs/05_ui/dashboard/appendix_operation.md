# 運用用ダッシュボード

本運用ページは異常検知、改善アクションの発見を目的としており、数値の比較や部署別の数値などの分析については、分析用ダッシュボードで行う。

## 全体像
[ KPIサマリー ]  
[ 利用状況 ] 
[ RAG精度・品質 ]  
[ コスト ]

## 各セクション詳細

### 共通定義
・　アクティブユーザー: メッセ―ジ送信が1回以上あるユーザー

### [KPIサマリー]
カードで表示。
1. DAU: 今日のアクティブユーザー数
2. RAG利用率: RAGモードONのメッセージ / 総メッセージ数
3. 利用継続率: 初回メッセ―ジ送信から7日以内/30日以内に再利用した人数
4. エラー率: すべてのエラー発生メッセージ数/総botメッセージ数

### [利用状況]
1. チャート。
x軸: 直近30日, y軸: メッセージ数, マウスオーバー: メッセージ数、アクティブユーザー数を表示するチャート  
目的: 急増、急減の以上に気づくためのチャート 

2. 部署別ランキング
メッセージ数が少ない下位5~10部署を表示  
目的: 使われていない部署を特定し、ヒアリングを行うきっかけとする

### [RAG精度]
カードで表示
1. RAGヒット率
2. 平均参照親チャンク数

詳細は分析ページで確認してもらうため、あくまでも簡単な数値だけ乗せておく

### [コスト]
1. 今月の推定コスト  
平均入出力トークン数から計算
